# App "Eco sounds"

Link to app: [https://nata-kostina.github.io/app-eco-sounds/](https://nata-kostina.github.io/app-eco-sounds/ "Eco-sounds")
***
## Description

Eco sounds is a simple application that plays bird's singing. A user can choose one of five birds and enjoy its singing looking at a nice picture on his screen.

Features:

* Choose a bird
* Play singning
* Change background picture
